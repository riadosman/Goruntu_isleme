Virtual environment nedir? Neden her projede lazım?

Gerçek hayat benzetmesi: "Her proje kendi bavulunu taşır" — Proje A'nın ihtiyacı numpy 1.24, Proje B'nin ihtiyacı numpy 1.26. İkisi aynı bilgisayarda çakışmasın diye her projeye ayrı bir "kutu" (venv) veriyoruz.

try/except ile sağlam kod yazma — graceful error handling

Sabah try/except Temelleri — 3.5 saat
2.1 try/except Temelleri 1.5 saat

Hata (Exception) nedir? Program çalışırken beklenmeyen bir şey olduğunda Python "hata fırlatır".

Gerçek hayat: Mutfakta yemek yapıyorsun, tuz bitti. 2 seçenek: (A) Mutfağı yak (programın çökmesi), (B) "Tuz yok, tuzsuz devam edeyim" de (try/except)

try:
    # Hata olabilecek kod
    sonuc = 10 / 0
except ZeroDivisionError:
    # Hata olursa burasi calisir
    print("Sifira bolme hatasi!")

Farklı hata türleri:

  • FileNotFoundError → Dosya bulunamadı
  • ValueError → Yanlış değer ("abc"yi int'e çevirme)
  • KeyError → Dictionary'de olmayan anahtar
  • TypeError → Yanlış tip ("5" + 5)
  • IndexError → Listede olmayan index
  • Exception → Tüm hataları yakalar (son çare)

Alıştırmalar (her biri ayrı try/except):

  1. Kullanıcıdan sayı al, 100'e böl. Sayı girmezse hata yakala.
  2. Bir dosya oku. Dosya yoksa "Dosya bulunamadı" yaz.
  3. Bir listeden index ile eleman al. Index yoksa hata yakala.
  4. Bir dictionary'den key ile değer al. Key yoksa hata yakala.
2.2 try/except/else/finally 1 saat
  • else → Hata olMAZSA çalışır
  • finally → Hata olsa da olmasa da MUTLAKA çalışır
  • finally ne işe yarar? Dosya kapatma, bağlantı kapatma gibi "temizlik" işleri
try:
    dosya = open("veri.txt", "r")
    icerik = dosya.read()
except FileNotFoundError:
    print("Dosya bulunamadi!")
else:
    print(f"Dosya basariyla okundu: {len(icerik)} karakter")
finally:
    print("Bu satir HER DURUMDA calisir")

Alıştırma: Kullanıcıdan dosya adı iste, dosyayı oku, satır sayısını yazdır. Her aşamada hata yönetimi olsun.

2.3 Birden Fazla except ve Hata Mesajları 1 saat
  • Bir try bloğunda birden fazla except olabilir
  • except Exception as e: ile hata mesajını yakalama
  • str(e) ile hata mesajını string'e çevirme
  • Kendi hata mesajları yazma: kullanıcıya anlamlı mesaj göster
try:
    deger = input("Sayi gir: ")
    sayi = int(deger)
    sonuc = 100 / sayi
except ValueError:
    print(f"'{deger}' bir sayi degil!")
except ZeroDivisionError:
    print("Sifira bolemezsin!")
except Exception as e:
    print(f"Beklenmeyen hata: {e}")
else:
    print(f"Sonuc: {sonuc:.2f}")

3.1 CSV Nedir? 1 saat

"Comma Separated Values" — virgülle ayrılmış değerler.

Excel'de açılabilen basit metin dosyası.

Gerçek hayat örneği: banka hesap özeti, öğrenci listesi, market fişi.

  • Neden CSV? Basit, her dilde okunabilir, Excel'de açılabilir, veritabanına aktarılabilir

Örnek CSV formatı:

isim,yas,sehir

3.2 Python csv Modülü — Yazma 1.5 saat
  • import csv
  • csv.writer ile satır satır yazma
  • Dosya açma modları: "w" (yeni yaz — eski veriyi siler!), "a" (append — sonuna ekler)
  • with open(...) as f: yapısı (context manager) — dosyayı güvenli aç/kapa. with bloğu bitince dosya otomatik kapanır.
  • newline="" parametresi neden lazım? (Windows'ta boş satır problemi)
import csv

# Yeni CSV dosyasi olustur
with open("ogrenciler.csv", "w", newline="") as dosya:
    yazici = csv.writer(dosya)
    yazici.writerow(["isim", "yas", "not"])  # Baslik satiri
    yazici.writerow(["Taha", 23, 85])
    yazici.writerow(["Ali", 25, 72])
    yazici.writerow(["Ayse", 22, 95])

print("CSV dosyasi olusturuldu!")

csv.DictWriter ile sütun başlıklı yazma (daha okunabilir):

import csv

basliklar = ["isim", "yas", "not"]

with open("ogrenciler2.csv", "w", newline="") as dosya:
    yazici = csv.DictWriter(dosya, fieldnames=basliklar)
    yazici.writeheader()

3.3 Python csv Modülü — Okuma 1 saat
  • csv.reader ile satır satır okuma
  • csv.DictReader ile sütun başlıklı okuma (her satır bir dictionary)
import csv

# csv.reader ile okuma
with open("ogrenciler.csv", "r") as dosya:
    okuyucu = csv.reader(dosya)
    basliklar = next(okuyucu)  # Ilk satir baslik
    for satir in okuyucu:
        print(f"Isim: {satir[0]}, Yas: {satir[1]}, Not: {satir[2]}")

# csv.DictReader ile okuma (daha okunakli)
with open("ogrenciler.csv", "r") as dosya:
    okuyucu = csv.DictReader(dosya)
    for satir in okuyucu:
        print(f"Isim: {satir['isim']}, Not: {satir['not']}")

3.4 İhlal Kayıt Sistemi Tasarımı 1.5 saat

GuardWatch'a CSV kayıt sistemi ekle. Her ihlal olduğunda bir satır ekle.

CSV formatı: zaman, kisi_id, ihlal_turu, sure_sn, ear_degeri

import csv
from datetime import datetime

def ihlal_kaydet(kisi_id, ihlal_turu, sure, ear):
    """Ihlali CSV dosyasina kaydet"""
    with open("ihlaller.csv", "a", newline="") as dosya:
        yazici = csv.writer(dosya)
        zaman = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        yazici.writerow([zaman, kisi_id, ihlal_turu, sure, f"{ear:.3f}"])

# Kullanim:
ihlal_kaydet("KID_1", "goz_kapali", 3.5, 0.18)
ihlal_kaydet("KID_2", "hareketsiz", 12.0, 0.25)

3.5 TEKRAR — time.time() ile CSV Timestamp 1 saat

Tekrar konusu: time.time() ve datetime birlikte kullanımı.

  • Unix timestamp'ı okunabilir zamana çevirme: datetime.fromtimestamp()
  • CSV'ye yazılacak zaman formatı: %Y-%m-%d %H:%M:%S
import time
from datetime import datetime

# Unix timestamp
ts = time.time()
print(f"Unix timestamp: {ts}")  # 1741089600.123

# Okunabilir format
okunabilir = datetime.fromtimestamp(ts)
print(f"Okunabilir: {okunabilir}")  # 2026-03-04 14:30:00.123

# Formatli string
formatli = okunabilir.strftime("%Y-%m-%d %H:%M:%S")
print(f"Formatli: {formatli}")  # 2026-03-04 14:30:00

4.1 os Modülü Temel İşlemler 1.5 saat

import os — dosya ve klasör işlemleri için temel modül.

  • os.makedirs("klasor/alt_klasor", exist_ok=True) — klasör oluştur (varsa hata verme)
  • os.path.exists("dosya.txt") — dosya/klasör var mı kontrol et
  • os.listdir("klasor") — klasördeki dosyaları listele
  • os.path.join("klasor", "alt", "dosya.txt") — platform bağımsız yol birleştir
  • Neden önemli? Windows: klasor\dosya.txt, Mac/Linux: klasor/dosya.txt. os.path.join doğru olanı seçer.
  • os.remove("dosya.txt") — dosya silme
  • os.path.getsize("dosya.txt") — dosya boyutu (byte)
import os

# Klasor olustur
os.makedirs("kayitlar/2026-03-05", exist_ok=True)

# Var mi kontrol et
if os.path.exists("kayitlar"):
    print("kayitlar klasoru mevcut!")

# Dosya listele
dosyalar = os.listdir("kayitlar")
print(f"Dosyalar: {dosyalar}")

# Yol birlestir
tam_yol = os.path.join("kayitlar", "2026-03-05", "ihlal_001.jpg")
print(f"Tam yol: {tam_yol}")

4.2 pathlib Modülü — Modern Alternatif 1 saat

from pathlib import Path — dosya yollarını nesne gibi kullanma.

4.3 cv2.imwrite() ile Frame Kaydetme 1 saat
  • cv2.imwrite("dosya.jpg", frame) — frame'i resim olarak kaydet
  • Dosya formatı uzantıya göre belirlenir: .jpg, .png
  • JPG: küçük dosya, kayıplı sıkıştırma. PNG: büyük dosya, kayıpsız.

import cv2
from datetime import datetime

def ihlal_frame_kaydet(frame, kisi_id, ear_degeri):
    """Ihlal anindaki frame'i kaydet"""
    # Tarih bazli klasor
    tarih = datetime.now().strftime("%Y-%m-%d")
    klasor = f"kayitlar/{tarih}"
    os.makedirs(klasor, exist_ok=True)

    # Dosya adi: ihlal_KID3_2026-03-05_14-30-05.jpg
    zaman = datetime.now().strftime("%Y-%m-%d_%H-%M-%S")
    dosya_adi = f"ihlal_{kisi_id}_{zaman}.jpg"
    tam_yol = os.path.join(klasor, dosya_adi)

    # Frame'e bilgi yaz
    bilgi = f"EAR: {ear_degeri:.3f} | {kisi_id} | {zaman}"
    cv2.putText(frame, bilgi, (10, 30), cv2.FONT_HERSHEY_SIMPLEX, 0.7, (0, 0, 255), 2)

    # Kaydet
    cv2.imwrite(tam_yol, frame)
    return tam_yol

4.5 Eski Kayıtları Temizleme 1 saat

X günden eski dosyaları otomatik silme.

  • os.path.getmtime() ile dosyanın son değiştirilme zamanı
  • time.time() - mtime > X_GUN * 86400 kontrolü
import os
import time

def eski_kayitlari_temizle(klasor, gun_limiti=7):
    """Belirtilen gun sayisindan eski dosyalari sil"""
    simdi = time.time()
    silinen = 0

    for dosya_adi in os.listdir(klasor):
        dosya_yolu = os.path.join(klasor, dosya_adi)
        if os.path.isfile(dosya_yolu):
            dosya_yasi_gun = (simdi - os.path.getmtime(dosya_yolu)) / 86400
            if dosya_yasi_gun > gun_limiti:
                os.remove(dosya_yolu)
                silinen += 1
                print(f"Silindi: {dosya_adi} ({dosya_yasi_gun:.1f} gun eski)")

    print(f"Toplam {silinen} eski dosya silindi.")
Dikkat: Dosya silme geri alınamaz! Test ederken önce print() ile hangi dosyaların silineceğini göster, emin olduktan sonra os.remove() satırını aç.

5.1 Threading Nedir? 1.5 saat

Ana program + yan iş aynı anda çalışma

Gerçek hayat: Yemek pişirirken çamaşır makinesini çalıştırma. İkisi aynı anda ilerliyor!

Normalde Python kodu satır satır sırayla çalışır. Threading ile iki iş “aynı anda” yapılabilir.

  • threading.Thread(target=fonksiyon) ile basit thread
  • .start() ile başlat, .join() ile bekle
  • Daemon thread: Ana program bitince otomatik kapanan thread
import threading
import time

def uzun_is():
    print("Uzun is basladi...")
    time.sleep(3)
    print("Uzun is bitti!")

def kisa_is():
    print("Kisa is basladi...")
    time.sleep(1)
    print("Kisa is bitti!")

# SIRAYLA (toplam ~4 saniye):
uzun_is()
kisa_is()

# THREAD ILE (toplam ~3 saniye):
t1 = threading.Thread(target=uzun_is)
t2 = threading.Thread(target=kisa_is)
t1.start()
t2.start()
t1.join()
t2.join()
print("Her iki is de bitti!")

Alıştırma: 3 farklı “iş” fonksiyonu yaz (her biri farklı süre beklesin). Önce sırayla, sonra thread ile çalıştır. Süre farkını ölç.

5.4 HTTP İstekleri — requests Kütüphanesi 1.5 saat

HTTP nedir? Web'in iletişim protokolü. Tarayıcın bir siteye girdiğinde HTTP isteği gönderir.

  • GET: Veri al (web sayfası oku, API'den bilgi çek)
  • POST: Veri gönder (form doldur, login yap)
  • pip install requests
  • requests.get(url) ile GET isteği
  • response.status_code: 200=başarılı, 404=bulunamadı, 500=sunucu hatası
  • response.json() ile JSON yanıtı parse et
  • response.text ile düz metin yanıt
import requests

# Basit GET istegi
response = requests.get("https://httpbin.org/get")
print(f"Status: {response.status_code}")  # 200
print(f"JSON: {response.json()}")

# Public API ornegi - IP bilgisi
response = requests.get("https://api.ipify.org?format=json")
veri = response.json()
print(f"IP adresin: {veri['ip']}")

Status kodları:

  • 200: OK — başarılı
  • 404: Not Found — sayfa/kaynak bulunamadı
  • 500: Internal Server Error — sunucu hatası
  • 403: Forbidden — erişim yasak
  • timeout: Sunucu yanıt vermedi

5.5 Hata Yönetimi ile HTTP 1 saat

İnternet bağlantısı yoksa ne olur? Sunucu cevap vermezse ne olur?

  • timeout parametresi: requests.get(url, timeout=5) — 5 saniye bekle, cevap gelmezse hata
  • try/except ile HTTP hata yönetimi
import requests

def guvenli_istek(url, timeout=5):
    """HTTP istegi yap, hata olursa None don"""
    try:
        response = requests.get(url, timeout=timeout)
        response.raise_for_status()  # 4xx/5xx ise hata firlat
        return response.json()
    except requests.exceptions.Timeout:
        print(f"UYARI: {url} zaman asimi!")
        return None
    except requests.exceptions.ConnectionError:
        print(f"UYARI: {url} baglanti hatasi!")
        return None
    except requests.exceptions.HTTPError as e:
        print(f"UYARI: HTTP hatasi: {e}")
        return None

# Kullanim:
veri = guvenli_istek("https://api.ipify.org?format=json")
if veri:
    print(f"IP: {veri['ip']}")
else:
    print("Veri alinamadi!")
